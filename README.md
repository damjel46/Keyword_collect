# 미리캔버스 태그 생성기

미리캔버스(Miricanvas) 요소(아이콘/일러스트) 투고 시 붙일 태그를 이름과 사진만으로 추천해주는 도구입니다.

- 서버 없이 `index.html` 파일 하나로 완전히 로컬에서 동작합니다.
- 입력한 이름/사진은 어디로도 전송되지 않고, 이 브라우저 안에서만 처리됩니다.
- PC와 모바일 브라우저 모두에서 그대로 열립니다.

## 온라인으로 바로 쓰기

👉 **[여기를 클릭해서 바로 사용하기](GITHUB_PAGES_URL_여기에_넣기)**

(저장소 소유자가 GitHub Pages를 켜면 이 링크로 설치 없이 바로 사용할 수 있어요.)

## 내 컴퓨터/휴대폰에 받아서 쓰기

온라인 링크 없이 파일만 따로 받아서 써도 똑같이 동작합니다.

1. 이 저장소에서 [`index.html`](index.html) 파일을 다운로드합니다. (저장소 페이지에서 `index.html` 클릭 → 우측 상단 다운로드 아이콘)
2. 다운로드한 `index.html`을 더블클릭(또는 모바일에서는 다운로드 후 파일 앱에서 열기)하면 브라우저에서 바로 실행됩니다.
3. 파일 하나만 있으면 되고, 인터넷 연결 없이도 그대로 동작합니다.

## 사용법

1. **요소 이름 입력** — 태그를 만들고 싶은 요소의 이름을 적습니다. (예: 과학, 비커, 강아지)
2. **이미지 첨부 (선택)** — 사진을 첨부하면 그림 스타일(귀여운/심플 vs 사실적/디테일)까지 분석해서 태그에 반영합니다.
3. **태그 생성 버튼 클릭** — 이름과 일치하는 카테고리를 찾아 관련 태그 목록을 만들어줍니다.
4. **태그 클릭해서 담기** — 원하는 태그를 클릭하면 하단 "담아둔 태그"에 콤마로 구분되어 모입니다.
5. **복사하기** — 담아둔 태그를 클립보드에 복사해서 미리캔버스 업로드 화면에 바로 붙여넣으세요.
6. **CSV/JSON으로 내보내기** — 지금까지 만든 항목 전체를 파일로 저장할 수 있습니다.
7. **전체 삭제** — 저장된 항목을 모두 지웁니다.

## 태그가 안 나올 때 (매칭 실패)

내부에 등록된 카테고리 사전에 없는 이름을 입력하면 태그가 생성되지 않습니다. 이 경우:

- 비슷한 뜻의 다른 이름으로 다시 시도해보세요. (예: "냥이" 대신 "고양이")
- 카테고리를 직접 추가하고 싶다면 `index.html` 안의 `CATEGORY_KEYWORDS`, `SYNONYMS` 객체를 편집하면 됩니다. (코드 상단 `<script>` 태그 안에 있습니다)

## 기술적으로 궁금하다면

- 이름 매칭과 그림 스타일 분석 로직 모두 `index.html` 안에 순수 JavaScript로 들어있습니다. 별도 서버, API 키, 외부 라이브러리가 전혀 없습니다.
- 만든 항목은 브라우저의 `localStorage`에만 저장됩니다. 다른 기기나 브라우저에서는 보이지 않습니다.
